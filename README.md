# CUDA_C-CPP_with_Python
Interfacing C/C++ CUDA code with Python

Run demo as:
    
    $ python cuda_test.py
   
In order to run this demo your system must have following packages:
    
    Python(2.7.13)
    OpenCV(>=2.4.x)
    Numpy(1.11.3)
    nvcc(>7.0)
